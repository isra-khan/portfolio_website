### Getting started

1. Install dependencies:

   ```bash
   cd /Users/israkhan/website
   npm install
   ```

2. Run the dev server:

   ```bash
   npm run dev
   ```

3. Open the printed local URL (usually `http://localhost:5173`) in your browser.

### Customization notes

- Replace the GitHub and LinkedIn URLs in `src/App.jsx` with your real profiles.
- Update the dates and details in the **Experience** and **Education** sections.
- Add real testimonials and blog posts when you have them.
- To make the **Download Resume** button work, place a PDF in the public folder and turn the button into a link to that file.
